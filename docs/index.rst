Jacc-hammer
===========

.. toctree::
    :hidden:
    :maxdepth: 1

    reference


Fuzzy-matching for larger data-sets.

Installation
------------

.. code-block:: console

    $ pip install git+ssh://git@github.com/nestauk/jacc-hammer.git


Usage
-----

.. code-block:: console

    $ jacc-hammer [OPTIONS]

.. option:: --help

    Display help information and exit.

.. option:: --version

    Display the version and exit.
